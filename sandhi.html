सन्धिः

Rule-application notes for sound changes. The Maheshwara Sutras page keeps the sound groups and प्रत्याहार notation; this page keeps the places where those sound groups change in use.

Study Map

Sandhi notes are grouped by the kind of sound being changed. Vowel and visarga sandhi are current; consonant sandhi is the next area to grow.

अच्सन्धिः

Vowel sandhi: one vowel meets another vowel.

अच् = vowels
अच् + अच् -> vowel sandhi rules

  • यान्तवान्तसन्धिः
  • पूर्वरूप, पररूप, प्रकृतिभाव

विसर्गसन्धिः

विसर्गसन्धिः changes a final ः. Check both sides: the sound before ः, and the first sound of the next word.

अः -> ओ

अः + अ -> ओऽ
अः + soft consonant/semivowel -> ओ + soft consonant/semivowel
  • गजः + अगच्छत् -> गजोऽगच्छत्
  • बालः + रमते -> बालो रमते
  • नः + मित्रः -> नो मित्रः

विसर्गस्य लोपः

अः + vowel other than अ -> अ + vowel
आः + vowel or soft consonant/semivowel -> आ + vowel or soft consonant/semivowel
सः / एषः + sound other than अ -> स / एष + sound
  • रामः + इति -> राम इति
  • बालाः + आगच्छन्ति -> बाला आगच्छन्ति
  • बालाः + गच्छन्ति -> बाला गच्छन्ति
  • सः + पठति -> स पठति
  • एषः + गच्छति -> एष गच्छति

विसर्गस्य रत्वम्

not अः / not आः + soft consonant/semivowel -> र् + soft consonant/semivowel
not अः / not आः + vowel -> र् + vowel
  • हरिः + अवदत् -> हरिरवदत्
  • सिद्धिः + भवतु मे सदा -> सिद्धिर्भवतु मे सदा
  • गुरुः + ब्रह्मा -> गुरुर्ब्रह्मा
  • गुरुः + विष्णुः -> गुरुर्विष्णुः

श् / ष् / स् Replacements

ः + स् / त् / थ् -> स् + स् / त् / थ्
ः + श् / च् / छ् -> श् + श् / च् / छ्
ः + ष् / ट् / ठ् -> ष् + ष् / ट् / ठ्
  • नमः + ते -> नमस्ते
  • कः + चन -> कश्चन
  • रामः + षष्ठः -> रामष्षष्ठः

Special Visarga Sounds

ः + क् / ख् -> जिह्वामूलीयः
ः + प् / फ् -> उपध्मानीयः
  • रामः करोति: careful recitation uses जिह्वामूलीयः before क्
  • रामः पठति: careful recitation uses उपध्मानीयः before प्
  • These are often printed with the ordinary विसर्गः sign.

Reading Joined Forms

एकोऽश्वोऽवसत्
= एकः अश्वः अवसत्
= one horse lived
  • ऽ is अवग्रहः; it marks a missing short अ.
  • एकोऽश्वः keeps अश्वः hidden inside the joined form.

अच्सन्धिः

अच्सन्धिः means vowel sandhi. It happens when one vowel meets another vowel, usually at a word boundary or inside a compound.

सवर्णदीर्घसन्धिः

अष्टाध्यायी: ६.१.१०१ अकः सवर्णे दीर्घः 

अ + अ / आ -> आ
इ / ई + इ / ई -> ई
उ / ऊ + उ / ऊ -> ऊ
  • राम + अवतारः -> रामावतारः
  • गिरि + ईशः -> गिरीशः
  • गुरु + उपदेशः -> गुरूपदेशः

गुणसन्धिः

अष्टाध्यायी: ६.१.८७ आद्गुणः 

अ / आ + इ / ई -> ए
अ / आ + उ / ऊ -> ओ
अ / आ + ऋ / ॠ -> अर्
  • देव + इन्द्रः -> देवेन्द्रः
  • महा + ईश्वरः -> महेश्वरः
  • महा + उत्सवः -> महोत्सवः
  • महा + ऋषिः -> महर्षिः

वृद्धिसन्धिः

अष्टाध्यायी: ६.१.८८ वृद्धिरेचि 

अ / आ + ए / ऐ -> ऐ
अ / आ + ओ / औ -> औ
  • सदा + एव -> सदैव
  • तथा + एव -> तथैव
  • महा + ऐश्वर्यम् -> महैश्वर्यम्
  • महा + औषधम् -> महौषधम्

यण्सन्धिः

अष्टाध्यायी: ६.१.७७ इको यणचि 

इ / ई + vowel -> य् + vowel
उ / ऊ + vowel -> व् + vowel
ऋ / ॠ + vowel -> र् + vowel
ऌ + vowel -> ल् + vowel
  • यदि + अपि -> यद्यपि
  • इति + एवम् -> इत्येवम्
  • सु + आगतम् -> स्वागतम्
  • गुरु + आज्ञा -> गुर्वाज्ञा
  • पितृ + आदेशः -> पित्रादेशः

यान्तवान्तसन्धिः

अष्टाध्यायी: ६.१.७८ एचोऽयवायावः 

ए + vowel, except short अ -> अय् + vowel
ऐ + vowel -> आय् + vowel
ओ + vowel, except short अ -> अव् + vowel
औ + vowel -> आव् + vowel
  • द्वौ + अपि -> द्वावपि
  • सर्वे + आगताः -> सर्वयागताः
  • ए/ओ + short अ uses पूर्वरूपसन्धिः
  • Story words and meanings

Special Cases

अष्टाध्यायी: ६.१.१०९ एङः पदान्तादति, ६.१.८९ एत्येधत्यूठ्सु, १.१.११ ईदूदेद्द्विवचनं प्रगृह्यम्, ६.१.१२५ प्लुतप्रगृह्या अचि 

ए + short अ -> एऽ
ओ + short अ -> ओऽ
अ + ए -> ए
ऽ marks the dropped अ

  • पररूपसन्धिः: प्र + एधते -> प्रेधते
  • प्रकृतिभावः: द्वे + अपि -> द्वे अपि
