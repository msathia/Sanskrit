हलन्तशब्दाः

Short lesson notes for identifying consonant-ending stems and a few useful corrections from class practice.

Core Contrast

हलन्तशब्दः

हल् + अन्तः
= ending in a consonant

A हलन्तशब्दः is a word or stem whose base ends in a consonant.

अजन्तशब्दः

राम = अजन्तशब्दः, because it ends in vowel अ
जगत् = हलन्तशब्दः, because it ends in consonant त्

अन्तः कथं ज्ञेयः?

Identify the ending of the base stem, not always the last visible letter of a declined form.

जकारान्तः

वणिक् -> stem वणिज्
अन्तः = ज्

So this is जकारान्तः.

षकारान्तः

द्विषः -> stem द्विष्
अन्तः = ष्

So this is षकारान्तः.

नकारान्तः

कर्म -> stem कर्मन्
अन्तः = न्

Neuter words can hide the stem-final consonant in common forms.

सकारान्तः

वयांसि -> stem वयस्
अन्तः = स्

Use the stem to decide the अन्तः.

भवत्-शब्दः

सम्बोधनप्रथमा

हे भवन्
हे भवन्तौ
हे भवन्तः

So it is not correct to say that भवत्-शब्दः has no सम्बोधनप्रथमा.

Correct Sentence

भोः! भवतः नाम किम्?

Use नाम, not नामः, because नाम is नपुंसकलिङ्गम्.

Vocabulary

पद्यप्रवेशः

पद्य = verse / poetry
प्रवेशः = entry / introduction
पद्यप्रवेशः = introduction to poetry or verse
