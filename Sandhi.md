# Sandhi Notes

## अच्सन्धिः / अच्-सन्धिः

अच्सन्धिः or अच्-सन्धिः means vowel sandhi.

`अच्` means vowels:

```text
अ आ इ ई उ ऊ ऋ ॠ ऌ ए ऐ ओ औ
```

अच्सन्धिः happens when one vowel meets another vowel, usually at a word boundary or inside a compound.

Basic idea:

```text
final vowel + initial vowel -> changed vowel form
```

The change may produce a long vowel, गुण, वृद्धि, or a glide form depending on the vowel pair.

### सवर्णदीर्घसन्धिः

This is one of the first vowel-sandhi patterns. When same-family vowels meet, they combine into the corresponding long vowel.

```text
अ + अ / आ -> आ
इ / ई + इ / ई -> ई
उ / ऊ + उ / ऊ -> ऊ
```

Examples:

```text
राम + अवतारः -> रामावतारः
गिरि + ईशः -> गिरीशः
गुरु + उपदेशः -> गुरूपदेशः
```

Reading trick: when a sandhi form has a long vowel in the middle, ask which two vowels may have met there.

### गुणसन्धिः

गुणसन्धिः is a vowel-sandhi rule where final `अ` or `आ` combines with a following `इ/ई`, `उ/ऊ`, or `ऋ/ॠ` and produces the corresponding गुण form.

Rule table:

```text
अ / आ + इ / ई -> ए
अ / आ + उ / ऊ -> ओ
अ / आ + ऋ / ॠ -> अर्
```

Examples:

```text
देव + इन्द्रः -> देवेन्द्रः
महा + ईश्वरः -> महेश्वरः
महा + उत्सवः -> महोत्सवः
महा + ऋषिः -> महर्षिः
```

Reading trick: if a sandhi form has `ए`, `ओ`, or `अर्`, check whether it may have come from `अ/आ` followed by `इ/ई`, `उ/ऊ`, or `ऋ/ॠ`.

### वृद्धिसन्धिः

वृद्धिसन्धिः is a vowel-sandhi rule where final `अ` or `आ` combines with a following `ए/ऐ` or `ओ/औ` and produces the corresponding वृद्धि form.

Rule table:

```text
अ / आ + ए / ऐ -> ऐ
अ / आ + ओ / औ -> औ
```

Examples:

```text
सदा + एव -> सदैव
तथा + एव -> तथैव
महा + ऐश्वर्यम् -> महैश्वर्यम्
महा + औषधम् -> महौषधम्
```

Reading trick: if a sandhi form has `ऐ` or `औ`, check whether it may have come from `अ/आ` followed by `ए/ऐ` or `ओ/औ`.

### यण्सन्धिः

यण्सन्धिः is a vowel-sandhi rule where final `इ/ई`, `उ/ऊ`, `ऋ/ॠ`, or `ऌ` changes into the matching semivowel before another vowel.

Rule table:

```text
इ / ई + vowel -> य् + vowel
उ / ऊ + vowel -> व् + vowel
ऋ / ॠ + vowel -> र् + vowel
ऌ + vowel -> ल् + vowel
```

Examples:

```text
यदि + अपि -> यद्यपि
इति + एवम् -> इत्येवम्
सु + आगतम् -> स्वागतम्
गुरु + आज्ञा -> गुर्वाज्ञा
पितृ + आदेशः -> पित्रादेशः
```

Reading trick: if a form has `य्`, `व्`, or `र्` between two vowel sounds, check whether it may have come from `इ/ई`, `उ/ऊ`, or `ऋ/ॠ` before a vowel. If the two vowels are same-family, first check सवर्णदीर्घसन्धिः.

### यान्तवान्तसन्धिः

यान्तवान्तसन्धिः is a vowel-sandhi rule where final `ए`, `ऐ`, `ओ`, or `औ` changes before another vowel.

Rule table:

```text
ए + vowel, except short अ -> अय् + vowel
ऐ + vowel -> आय् + vowel
ओ + vowel, except short अ -> अव् + vowel
औ + vowel -> आव् + vowel
```

Examples:

```text
द्वौ + अपि -> द्वावपि
सर्वे + आगताः -> सर्वयागताः
मुक्त्यै + एव -> मुक्त्यायेव
राजानौ + अपि -> राजानावपि
```

Important exception: `ए/ओ + short अ` uses पूर्वरूपसन्धिः, not यान्तवान्तसन्धिः.

### Special Cases

These rules are useful because they block or override an expected general vowel-sandhi rule.

#### पूर्वरूपसन्धिः

```text
ए + short अ -> एऽ
ओ + short अ -> ओऽ
```

`ऽ` is called अवग्रहः. It marks that the following short `अ` has disappeared.

Examples:

```text
हरे + अत्र -> हरेऽत्र
विष्णो + अत्र -> विष्णोऽत्र
अहो + अद्भुतम् -> अहोऽद्भुतम्
```

#### पररूपसन्धिः

In special prefix-like contexts, the later vowel remains instead of the expected वृद्धि form.

```text
प्र + एधते -> प्रेधते
```

Expected by वृद्धिसन्धिः:

```text
प्र + एधते -> प्रैधते
```

Actual by पररूपसन्धिः:

```text
प्र + एधते -> प्रेधते
```

#### प्रकृतिभावः

प्रकृतिभावः means the original form is kept. Expected sandhi is blocked.

Story examples:

```text
द्वे + अपि -> द्वे अपि
मती + एते -> मती एते
श्रीपते३ + अत्र -> श्रीपते३ अत्र
```

Practice examples:

```text
हरी + एतौ -> हरी एतौ
विष्णू + इमौ -> विष्णू इमौ
गङ्गे + अमू -> गङ्गे अमू
वन्देते + इमौ -> वन्देते इमौ
```
