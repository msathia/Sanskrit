यान्तवान्तसन्धिः

Story words formed when final ए, ऐ, ओ, or औ changes before another vowel. For ए/ओ + short अ, use पूर्वरूपसन्धिः instead. Each entry keeps only the split, sound change, final form, and English meaning.

Rule

यान्तवान्तसन्धिः

ए + vowel, except short अ -> अय् + vowel
ऐ + vowel -> आय् + vowel
ओ + vowel, except short अ -> अव् + vowel
औ + vowel -> आव् + vowel

Story Words

राजानावपालयताम्

राजानौ + अपालयताम्
औ + अ -> आव् + अ
...नौ + अ...
-> ...नाव् + अ...
-> राजानावपालयताम्

The two kings ruled or protected.

द्वावपि

औ + अ -> आव् + अ
...द्वौ + अ...
-> ...द्वाव् + अ...
-> द्वावपि

Both; the two also.
