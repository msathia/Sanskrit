पूर्वरूपसन्धिः

Notes for the special case where final ए or ओ meets short अ. The short अ disappears and is marked with अवग्रहः.

Rule

पूर्वरूपसन्धिः

ए + short अ -> एऽ
ओ + short अ -> ओऽ

The sign ऽ is called अवग्रहः (avagrahaḥ). It marks that the following short अ has disappeared.

Practical Distinction

ए + अ -> एऽ        पूर्वरूपसन्धिः
ओ + अ -> ओऽ        पूर्वरूपसन्धिः

ऐ + अ -> आय् + अ    यान्तवान्तसन्धिः
औ + अ -> आव् + अ    यान्तवान्तसन्धिः

This is why सर्वे + अपि becomes सर्वेऽपि, but राजानौ + अपालयताम् becomes राजानावपालयताम्. The first ends in ए; the second ends in औ, not ओ.

Examples

सर्वेऽपि

सर्वे + अपि
ए + अ -> एऽ
...र्वे + अ...
-> ...र्वेऽ...
-> सर्वेऽपि

All also; all too.

तेऽपि

ते + अपि
ए + अ -> एऽ
...ते + अ...
-> ...तेऽ...
-> तेऽपि

They also; those also.

कोऽपि

को + अपि
ओ + अ -> ओऽ
को + अपि
-> कोऽपि

Someone; anyone.

गुरोऽत्र

गुरो + अत्र
ओ + अ -> ओऽ
...रो + अ...
-> ...रोऽ...
-> गुरोऽत्र

O teacher, here.

हरेऽत्र

हरे + अत्र
ए + अ -> एऽ
...रे + अ...
-> ...रेऽ...
-> हरेऽत्र

O Hari, here. In study notes, prefer हरेऽत्र; without अवग्रहः it may look like हरेत्र.

अहोऽद्भुतम्

अहो + अद्भुतम्
ओ + अ -> ओऽ
...हो + अ...
-> ...होऽ...
-> अहोऽद्भुतम्

Ah, wonderful; oh, amazing. Use अहोऽद्भुतम्, not अहोद्भुतम्, when marking this sandhi.

राजानावपालयताम्

राजानौ + अपालयताम्
औ + अ -> आव् + अ
...नौ + अ...
-> ...नाव् + अ...
-> राजानावपालयताम्

Contrast example: this is यान्तवान्तसन्धिः because राजानौ ends in औ, not ओ.

Reading Tip

When you see ऽ, restore a short अ at that point while splitting the words.

सर्वेऽपि -> सर्वे + अपि
कोऽपि -> को + अपि
अहोऽद्भुतम् -> अहो + अद्भुतम्
