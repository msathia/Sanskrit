अच्सन्धिः Special Cases

Exceptions and blocked sandhi cases from the story: पूर्वरूपसन्धिः, पररूपसन्धिः, and प्रकृतिभावः.

पूर्वरूपसन्धिः

पूर्वरूपसन्धिः

ए + short अ -> एऽ
ओ + short अ -> ओऽ

The sign ऽ is called अवग्रहः (avagrahaḥ). It marks that the following short अ has disappeared.

Practical Distinction

ए + अ -> एऽ        पूर्वरूपसन्धिः
ओ + अ -> ओऽ        पूर्वरूपसन्धिः

ऐ + अ -> आय् + अ    यान्तवान्तसन्धिः
औ + अ -> आव् + अ    यान्तवान्तसन्धिः

This is why सर्वे + अपि becomes सर्वेऽपि, but राजानौ + अपालयताम् becomes राजानावपालयताम्. The first ends in ए; the second ends in औ, not ओ.

Examples

सर्वेऽपि

सर्वे + अपि
ए + अ -> एऽ
...र्वे + अ...
-> ...र्वेऽ...
-> सर्वेऽपि

All also; all too.

तेऽपि

ते + अपि
ए + अ -> एऽ
...ते + अ...
-> ...तेऽ...
-> तेऽपि

They also; those also.

कोऽपि

को + अपि
ओ + अ -> ओऽ
को + अपि
-> कोऽपि

Someone; anyone.

गुरोऽत्र

गुरो + अत्र
ओ + अ -> ओऽ
...रो + अ...
-> ...रोऽ...
-> गुरोऽत्र

O teacher, here.

हरेऽत्र

हरे + अत्र
ए + अ -> एऽ
...रे + अ...
-> ...रेऽ...
-> हरेऽत्र

O Hari, here. In study notes, prefer हरेऽत्र; without अवग्रहः it may look like हरेत्र.

विष्णोऽत्र

विष्णो + अत्र
ओ + अ -> ओऽ
...ष्णो + अ...
-> ...ष्णोऽ...
-> विष्णोऽत्र

O Vishnu, here. This is another story example where अवग्रहः marks the dropped short अ.

अहोऽद्भुतम्

अहो + अद्भुतम्
ओ + अ -> ओऽ
...हो + अ...
-> ...होऽ...
-> अहोऽद्भुतम्

Ah, wonderful; oh, amazing. Use अहोऽद्भुतम्, not अहोद्भुतम्, when marking this sandhi.

Reading Tip

When you see ऽ, restore a short अ at that point while splitting the words.

सर्वेऽपि -> सर्वे + अपि
कोऽपि -> को + अपि
अहोऽद्भुतम् -> अहो + अद्भुतम्

पररूपसन्धिः

Later Form Wins

In special prefix-like contexts, the expected वृद्धिसन्धिः is blocked and the later vowel remains. This is why it feels like an exception to the regular वृद्धि rule.

अ + ए -> ए
अ + ओ -> ओ

प्रेधते

प्र + एधते
अ + ए -> ए

Expected by वृद्धिसन्धिः:
प्र + एधते -> प्रैधते

Actual by पररूपसन्धिः:
प्र + एधते -> प्रेधते

In the story: तस्य तपसः तीक्ष्णता प्रेधते means “the intensity of his penance increases.”

प्रकृतिभावः

Original Form Is Kept

Here the expected sandhi is blocked. The word remains in its original form, often because the form is dual or because a calling form has a pluta vowel.

द्वे अपि

द्वे + अपि

Expected by पूर्वरूपसन्धिः:
ए + अ -> एऽ
द्वे + अपि -> द्वेऽपि

Actual by प्रकृतिभावः:
द्वे + अपि -> द्वे अपि

Meaning: “both also.” Here द्वे is a dual form.

मती एते

मती + एते

Expected by यण्सन्धिः:
ई + ए -> य् + ए
मती + एते -> मत्येते

Actual by प्रकृतिभावः:
मती + एते -> मती एते

Meaning: “these two thoughts.” Here मती is a dual form.

श्रीपते३ अत्र

श्रीपते३ + अत्र

Expected by पूर्वरूपसन्धिः:
ए + अ -> एऽ
श्रीपते + अत्र -> श्रीपतेऽत्र

Actual by प्रकृतिभावः:
श्रीपते३ + अत्र -> श्रीपते३ अत्र

Meaning: “O Lord of Lakshmi, come here.” The ३ marks a prolonged calling vowel, so the form is kept.

हरी एतौ

हरी + एतौ

Expected by यण्सन्धिः:
ई + ए -> य् + ए
हरी + एतौ -> हर्येतौ

Actual by प्रकृतिभावः:
हरी + एतौ -> हरी एतौ

Practice example: dual form ending in ई.

विष्णू इमौ

विष्णू + इमौ

Expected by यण्सन्धिः:
ऊ + इ -> व् + इ
विष्णू + इमौ -> विष्ण्विमौ

Actual by प्रकृतिभावः:
विष्णू + इमौ -> विष्णू इमौ

Practice example: dual form ending in ऊ.

गङ्गे अमू

गङ्गे + अमू

Expected by पूर्वरूपसन्धिः:
ए + अ -> एऽ
गङ्गे + अमू -> गङ्गेऽमू

Actual by प्रकृतिभावः:
गङ्गे + अमू -> गङ्गे अमू

Practice example: dual form ending in ए.

वन्देते इमौ

वन्देते + इमौ

Expected by यान्तवान्तसन्धिः:
ए + इ -> अय् + इ
वन्देते + इमौ -> वन्देतयिमौ

Actual by प्रकृतिभावः:
वन्देते + इमौ -> वन्देते इमौ

Practice example: dual verb form ending in ए.
