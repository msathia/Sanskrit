अच्सन्धिः / अच्-सन्धिः

अच्सन्धिः means vowel sandhi. It happens when one vowel meets another vowel, usually at a word boundary or inside a compound.

सवर्णदीर्घसन्धिः

अ + अ / आ -> आ
इ / ई + इ / ई -> ई
उ / ऊ + उ / ऊ -> ऊ
  • राम + अवतारः -> रामावतारः
  • गिरि + ईशः -> गिरीशः
  • गुरु + उपदेशः -> गुरूपदेशः

गुणसन्धिः

अ / आ + इ / ई -> ए
अ / आ + उ / ऊ -> ओ
अ / आ + ऋ / ॠ -> अर्
  • देव + इन्द्रः -> देवेन्द्रः
  • महा + ईश्वरः -> महेश्वरः
  • महा + उत्सवः -> महोत्सवः
  • महा + ऋषिः -> महर्षिः

वृद्धिसन्धिः

अ / आ + ए / ऐ -> ऐ
अ / आ + ओ / औ -> औ
  • सदा + एव -> सदैव
  • तथा + एव -> तथैव
  • महा + ऐश्वर्यम् -> महैश्वर्यम्
  • महा + औषधम् -> महौषधम्

यण्सन्धिः

इ / ई + vowel -> य् + vowel
उ / ऊ + vowel -> व् + vowel
ऋ / ॠ + vowel -> र् + vowel
ऌ + vowel -> ल् + vowel
  • यदि + अपि -> यद्यपि
  • इति + एवम् -> इत्येवम्
  • सु + आगतम् -> स्वागतम्
  • गुरु + आज्ञा -> गुर्वाज्ञा
  • पितृ + आदेशः -> पित्रादेशः

यान्तवान्तसन्धिः

ए + vowel -> अय् + vowel
ऐ + vowel -> आय् + vowel
ओ + vowel -> अव् + vowel
औ + vowel -> आव् + vowel
  • द्वौ + अपि -> द्वावपि
  • सर्वे + आगताः -> सर्वयागताः
  • Story words and meanings

Study Workflow

Terminal learning works well when the locale is UTF-8 and the terminal font has good Devanagari support. When screenshots are useful, crop tightly and include the raw text too.

  • Prefer directly pasted Sanskrit text when possible.
  • Use screenshots for rendering issues, PDF snippets, handwritten notes, or layout questions.
  • Keep course PDFs, screenshots, and paid material out of Git.
  • Use Workflow.md for the detailed local workflow notes.
