शतृ-शानच्

Topic notes for present participles: verb forms such as “singing” or “while doing” that describe the subject while another action happens.

लट्लकारः

लट्लकारः is the present tense. These forms name the finite action in the sentence.

Present Forms

करोति = he/she/it does or makes
पचति = he/she/it cooks
गायति = he/she/it sings

These are लट्लकारः, प्रथमपुरुषः, एकवचनम्.

Person Memory

प्रथमपुरुषः = he/she/it/they
मध्यमपुरुषः = you
उत्तमपुरुषः = I/we

Participle Idea

A participle is a verb-derived form that often works like an adjective. In English, “singing” in “the singing boy” comes from a verb but describes the boy.

रामः गायन् पचति।
Rama cooks while singing.

Here गायन् comes from गायति, but it describes रामः.

Core Transformation

रामः गायति पचति च।

रामः गायति पचति च।
= Rama sings and cooks.

रामः गायन् पचति।

गायति -> गायन्
रामः गायन् पचति।
= Rama cooks while singing.

Tamil intuition: ராமன் பாடிக்கொண்டு சமைக்கிறான்.

शतृ / शानच्

Both are present participle प्रत्ययाः. They turn a present-tense verb idea into “doing” or “while doing.” The practical distinction comes from the verb class ending.

शतृ

Used only with परस्मैपदी verbs. A quick beginner memory is that many ति-ending present forms use शतृ.

परस्मैपदी धातुः -> शतृ

पचति -> पचन्
गायति -> गायन्
पठति -> पठन्

शानच्

Used only with आत्मनेपदी verbs. A quick beginner memory is that many ते-ending present forms use शानच्.

आत्मनेपदी धातुः -> शानच्

वन्दते -> वन्दमानः
लभते -> लभमानः
सेवते -> सेवमानः

Main Distinction

शतृ-प्रत्ययः -> only परस्मैपदीधातुः
शानच्-प्रत्ययः -> only आत्मनेपदीधातुः

ति-ending present forms -> often शतृ
ते-ending present forms -> often शानच्
  • गायति / पचति / पठति become गायन् / पचन् / पठन्.
  • वन्दते / लभते / सेवते become वन्दमानः / लभमानः / सेवमानः.

Word Examples

The examples below keep the same test: identify the present verb form first, then choose शतृ for परस्मैपदी and शानच् for आत्मनेपदी.

धरन्ती

धरन्ती is a feminine present participle. Think of it as an adjective-like form: “she who is holding, bearing, or carrying.”

धरति -> धरन् / धरन्ती

धरति = he/she/it holds, bears, carries
धरन् = holding / bearing, masculine
धरन्ती = holding / bearing, feminine

Because धरति is परस्मैपदी, it uses शतृ-प्रत्ययः, not शानच्.

धरन्तः

धरन्तः is also from धरति. The difference from धरन्ती is gender and number, not the प्रत्यय.

धरति -> धरन्तः / धरन्ती

धरन्तः = holding / bearing, masculine plural
धरन्ती = holding / bearing, feminine singular

Both are शतृ forms because धरति is परस्मैपदी.

पालयन्

पालयन् means “protecting” or “while protecting.” It describes a masculine noun doing the action.

पालयति -> पालयन्

पालयति = he/she/it protects
पालयन् = protecting / while protecting

राजा प्रजाः पालयन् तिष्ठति।
= The king remains while protecting the people.

Since पालयति is a ति-ending परस्मैपदी form, the participle uses शतृ. Tamil memory: காத்துக்கொண்டு.

अनुभवन्तः

अनुभवन्तः means “experiencing” or “feeling.” It is the masculine plural participle form.

अनुभवति -> अनुभवन्तः

अनुभवति = he/she/it experiences, feels
अनुभवन् = experiencing, masculine singular
अनुभवन्तः = experiencing, masculine plural

Because अनुभवति is a ति-ending परस्मैपदी form, this is शतृ, not शानच्.

डयमानः

डयमानः means “flying” or “while flying.” It is a useful contrast because the base verb ends in ते.

डयते -> डयमानः

डयते = he/she/it flies
डयमानः = flying / while flying

Since डयते is a ते-ending आत्मनेपदी form, the participle uses शानच्. Do not confuse डयते “flies” with दयते “shows compassion.”

आरभमाणस्य

आरभमाणस्य means “of the thing beginning” or “of the thing being started.” It is the genitive form of a शानच् participle.

आरभते -> आरभमाण- -> आरभमाणस्य

आरभते = he/she/it begins, starts
आरभमाण- = beginning / being started
आरभमाणस्य = of the thing being started

Since आरभते is a ते-ending आत्मनेपदी form, the participle uses शानच्. The ending -स्य gives the “of” meaning and can match a word like गीतस्य.

Meaning Examples

बालकः पठन् हसति।

पठति -> पठन्
बालकः पठन् हसति।
= The boy laughs while reading.

भक्तः वन्दमानः तिष्ठति।

वन्दते -> वन्दमानः
भक्तः वन्दमानः तिष्ठति।
= The devotee stands while saluting.
